CredFin

CredFin is a microcredit service that allows you to quickly get a loan for a small amount of money. Financial projects are always complicated, especially when there is a risk of getting a disadvantaged client. But after a thorough check, we were convinced of the service and the team behind it.

Challenge

Creation of an automatic online broker to collect and process online applications with the ability to flexibly and quickly set up business logic. We needed to create

  • Implementation of the service for data exchange with partners (financial institutions)
  • Optimization of automatic requests processing
  • Integration with business process management system

Results:

In order to maximize processing of all incoming requests, CredFin system can not only accept but also send requests for processing to various partners. At the same time, system administrators, when a new partner appears, can configure sending requests to their system using the built-in API request builder.

The main data storage base is - fdForge. All information about users, which was received from applications and partners, is accumulated and transferred to the CRM system.

Work done:

To successfully implement the main idea of the project, we decomposed all the tasks into a set of point problems, to each of which a personal approach was applied to find the optimal solution.
