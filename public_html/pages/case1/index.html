Mobile development, product design

ThatHLTH

ThatHLTH is a mobile app designed to support a healthy lifestyle. Its main function is to provide useful tips on how to monitor your health and keep it in good shape. The functionality is based on the user's biometric data and lifestyle.

Challenge

Our main goal was to create an app that combines a fitness tracker, an information guide on human health, and a social network with game elements. So we decided to break the work down into 3 main parts:

  • Discovery and Research
  • UX prototypes
  • User interface implementation

Results:

  • Home page that provides access to the quests and blog.
  • Ability to track activity by connecting fitness trackers and other devices.
  • Information about health indicators, vaccine list, allergies, and so on.
  • Quick access to chat with other users.

Work done:

More than 150 application screens have been created, including all possible states, pop-up screens with unique icons, and many settings screens.
